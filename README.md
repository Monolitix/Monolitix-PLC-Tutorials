# Monolitix PLC Tutorials

Welcome to the official tutorial repository for Monolitix PLC.

This repository contains the project files and source code used in the official Monolitix YouTube tutorials, helping users follow along step-by-step while learning how to develop and program Monolitix PLCs.

## Each folder contains the code used in a specific YouTube tutorial:

`Tutorial_1_First_Project` → Introduction and first PLC project
→ More tutorials will be added regularly

## Requirements

To follow these tutorials, you will need:

- `Visual Studio Code`
- `VSC Monolitix Extension`
- `Monolitix PLC`
- `USB-C Cable`

## Support

For documentation, updates, and more resources, visit the official Monolitix platform.
New tutorials are published regularly on the Monolitix YouTube channel.
